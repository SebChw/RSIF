Welcome to Generalized Isolation Forest's documentation!
========================================================

**Generalized Isolation Forest** (GIF) is a Python library implementing algorithm having the same name. This is the merge of 
**Isolation Forest**, **Similarity Forest** and **Random Similarity Forest**. We took some ideas from all of these to create GIF.
